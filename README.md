# TorchTEM

This repository contains `torchtem`, a fully differentiable simulator for transmission electron microscopes (TEM). It has been inspired by [abTEM](https://github.com/abTEM/abTEM), an ab inito transmission electron microscope simulator built by a [group at University of Vienna](https://www.mostlyphysics.net/).

The main goal was to keep the whole implementation fully differentiable to allow backpropagation through the models.
